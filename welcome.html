The Access To .Net Forms Converter

This project converts your MS Access Forms (any version) to .NET Forms or User Controls. *It supports conversion of MS Access Forms to Windows Forms/User Controls, or ASP.NET Forms/User Controls.* This is a Visual Basic .Net project, developed with Visual Studio .net 2012 and requires .Net 4.
